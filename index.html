Web 1.0（信息共享）
此时是静态技术阶段

是Web向用户展示信息的最有效载体

具有代表性的网站有：

新浪、 搜狐、 雅虎、 谷歌、 网易等

Web 2.0（信息共建）
此时是动态技术阶段

随着AJAX的出现

加强了网站与用户之间的互动

网站内容基于用户提供

网站的诸多功能也由用户参与建设

实现了网站与用户的双向互动交流

典型的代表有：

博客、 天涯、 豆瓣、 贴吧等

先了解一下最流行的技术栈

VueJS

前端新秀

（简化版）AngularJS双向绑定 + （简化版）ReactJS的模块和组件化

使用 Virtual DOM

提供了响应式和组件化的视图组件

将注意力集中保持在核心库，而将其他功能如路由和全局状态管理交给相关的库

官网：http://vuejs.org

ReactJS

采用虚拟节点( virtual dom )技术

jsx技术（XML语法），一切都是 JavaScript, HTML写入JS中

现在也越来越多地将 CSS 也纳入到 JavaScript 中来处理

以使用完整的编程语言 JavaScript 功能来构建视图页面

官网：https://facebook.github.io/react/
